# Wordle as React Component

### [Demo](https://haikelfazzani.github.io/react-wordle)

# Installation
```shell
$ npm install react-wordle
# yarn add react-wordle
```

# Example
```jsx
import { Wordle } from 'react-wordle';

<Wordle wordList={['piano', 'hello']} solution="piano" />
```
